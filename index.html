    My Projects

    MicroCue

    This project was built in response to a need at HackerYou for a way to organize those waiting in line to use the microwave. It was created using React, Firebase and the Giphy API. Users log in using Google Authentication, and from there are able to add themselves to the cue.

    World Of Flavour

    This project was created using HTML5, CSS3, Sass Javascript, jQuery and the Zomato API. It creates an interactive user experience where users can discover restaurants in the Toronto area that serve cuisines inspired by the continents of the world.

    EmojiMatch

    This pair programming project with Hao Wu utilizes Javascript and jQuery to create an engaging Emoji match game. The score is calculated and displayed dynamically to the user.

    Weather We Eat?
